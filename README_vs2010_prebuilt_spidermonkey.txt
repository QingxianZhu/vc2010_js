This archive contains prebuilt SpiderMonkey libraries built with Visual Studio 2010, 
and the associated header files.  This package is partial and non-official.

  js32d.lib 32 bit debug lib (/MD assumed)
  js32r.lib 32 bit release lib (/MT assumed)
  js64d.lib 64 bit debug lib (/MD assumed)
  js64r.lib 64 bit release lib (/MT assumed)

See also:

  http://www.mozilla.org/js/spidermonkey/release-notes/
